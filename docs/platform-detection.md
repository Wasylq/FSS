**If no match**: build a standalone scraper. Check the site for JSON APIs (browser DevTools → Network tab), sitemaps, or structured data before defaulting to HTML scraping.

**If you can't find any scrapeable endpoint after probing the obvious candidates** (`/`, `/sitemap.xml`, `/videos`, `/videos.php`, `/gallery`, `/scenes`, `/updates`, `/latest`, etc.) — **ask the user before going further**. The operator probably knows the working endpoint from their browser history or the site's own navigation, and one URL from them saves an open-ended search. Don't fall back to scraping the parent network's catalogue + client-side filter unless the operator confirms there's no per-site endpoint — that approach works but trades scrape time for design simplicity, and the operator should choose.

**Thin wrappers don't need tests** — only the `*util` package does. If the new scraper adds site-specific logic beyond config, add tests for that logic.

## Platform-specific notes

**Gamma segments:** Each segment's Algolia API key is isolated — keys from one segment can't query another. Decode the base64 API key to find `filters=segment%3A<name>`. Segments: `adulttime` (Adult Time network), `evilangel`, `evilangelnetwork`, `dfxtra` (Dogfart, 17 subsites), `openlife` (12 subsites), `zerotolerancefilms`, `addicted2girls`, `biphoria`, `blowpass`, `asgmax` (Next Door Studios family / gay all-access, 24 sub-sites). The API key is fetched from the site's `/en/videos` page with a `Referer` restriction. Sites needing a different Referer URL need `RefererBase` set in `gammautil.SiteConfig` — see the ASGMAX redirecting sites (marcusmojo / masonwyler / etc.) for the pattern.

**Aylo brand→domain mappings:** Multiple domains can share the same Aylo brand/instance. `letsdoeit` (letsdoeit.com), `men` (men.com + 7 alt domains: bigdicksatschool.com, drillmyhole.com, str8togay.com, thegayoffice.com, jizzorgy.com, menofuk.com, toptobottom.com), `metro` (shewillcheat.com, familyhookups.com, kinkyspa.com, girlgrind.com), `milehigh` (milfed.com), `killergram` (killergram.com), `bigstr` (czechhunter.com), `bromo` (bromo.com), `erito` (erito.com), `fakehub` (fakehub.com + 10 subsite domains — all redirect to fakehub.com except publicagent.com which is standalone), `twinkpop` (twinkpop.com), `whynotbi` (whynotbi.com — previously misconfigured as men.com).
